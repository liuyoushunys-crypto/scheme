# 多行字符串 `"""..."""` 支持

## 用法

Python 风格的三引号字符串，支持真实换行：

```scheme
;; 多行字符串数据
"""Hello
World!
This is Scheme."""

;; 多行 py 表达式
(py "np.array([
    [1, 2, 3],
    [4, 5, 6]
])")

;; 多行 f-string
(py-f """Values:
a = {a}
b = {b}
""")
```

## 实现

- **词法分析器** `parser/tokenize.py`：检测 `"""`，收集至闭合 `"""`，跳过首行换行
- **语法分析器** `parser/parse_form.py`：剥离三引号，构造 `Str` 值

## 注意

- 空内容 `""""""` 为有效空字符串
- `\"\"\"` 不可写在字符串中（转义未实现），需用 `"` 单行替代
